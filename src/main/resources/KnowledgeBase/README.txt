In this folder you can find our knowledge base, which is around 60% complete.
It sill needs little formatting and cleaning. Everything is subject to change.

Our pharmacy assistant system first receives the main problem of a client (Right now it can be only throat pain, diarrhea/vomiting, coughin)
Then it asks a series of questions and builds a confidence of recommendation for each of the available medicines.

For yes-no questions, the answers are represented as
1 : the medicine is good if the answer is yes
-1 : the medicine is bad if the answer is yes
0 : the answer is irrelevant
